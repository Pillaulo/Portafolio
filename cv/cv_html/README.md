# CV bilingüe en HTML

Currículum de Paulo Vicencio en español e inglés, sin LaTeX ni dependencias.

## Uso

1. Abre `index.html` en Chrome, Edge o Firefox.
2. Cambia el idioma con los botones `Español` / `English`.
3. Pulsa `Guardar como PDF`.
4. En el diálogo de impresión, selecciona:
   - Destino: `Guardar como PDF`
   - Papel: `A4`
   - Escala: `Predeterminada` o `100 %`
   - Gráficos de fondo: activados

## Edicion

- Contenido de ambos idiomas: `index.html`
- Colores, tipografía y diseño: `styles.css`
- Selector de idioma e impresión: `script.js`

Los archivos LaTeX originales extraídos se conservan en `source/es` y `source/en`
como respaldo.
